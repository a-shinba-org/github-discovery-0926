# 📸 写真共有アプリ (Photo Sharing App)

シンプルで使いやすい写真共有アプリケーションです。写真をアップロードして、美しいギャラリー形式で閲覧・管理できます。

## 🌟 主な機能

- **📤 写真アップロード**: ドラッグ&ドロップまたはファイル選択で簡単に写真をアップロード
- **🖼️ ギャラリー表示**: レスポンシブなグリッドレイアウトで写真を美しく表示
- **🔍 写真詳細表示**: 写真をクリックしてモーダルで詳細情報を確認
- **🗑️ 写真削除**: 不要な写真を削除する機能
- **💾 ローカル保存**: ブラウザのローカルストレージを使用してデータを保存
- **📱 レスポンシブデザイン**: モバイル・タブレット・デスクトップに対応

## 🚀 使い方

### 基本的な使い方

1. **写真のアップロード**
   - 上部のアップロードエリアに写真をドラッグ&ドロップ
   - または「ファイルを選択」ボタンをクリックして写真を選択
   - 複数の写真を同時にアップロード可能

2. **写真の閲覧**
   - ギャラリーで写真をクリックして詳細表示
   - モーダル内でファイル名、サイズ、アップロード日時を確認

3. **写真の削除**
   - 写真の詳細表示で「写真を削除」ボタンをクリック
   - 確認ダイアログが表示されるので「OK」で削除実行

### キーボードショートカット

モーダル表示中に以下のキーが使用できます：
- `←` / `→`: 前後の写真に移動
- `Esc`: モーダルを閉じる  
- `Delete`: 現在の写真を削除

## 🖥️ 動作環境

- **対応ブラウザ**: Chrome、Firefox、Safari、Edge（モダンブラウザ）
- **対応画像形式**: JPEG、PNG、GIF、WebP等（ブラウザがサポートする画像形式）
- **必要環境**: HTML5、CSS3、ES6対応ブラウザ

## 💻 開発・実行方法

### ローカル環境での実行

1. リポジトリをクローン：
   ```bash
   git clone https://github.com/a-shinba-org/github-discovery-0926.git
   cd github-discovery-0926
   ```

2. ウェブサーバーで実行：
   ```bash
   # Python3を使用する場合
   python -m http.server 8000
   
   # Node.jsのhttp-serverを使用する場合
   npx http-server .
   
   # PHP内蔵サーバーを使用する場合  
   php -S localhost:8000
   ```

3. ブラウザで `http://localhost:8000` にアクセス

### 直接ファイル実行

ローカルファイルとして `index.html` をブラウザで直接開くことも可能ですが、ファイルアップロード機能を完全に利用するためにはHTTPサーバー経由での実行を推奨します。

## 🏗️ 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: Flexbox/Grid、CSS Custom Properties、アニメーション
- **JavaScript (ES6+)**: クラスベースの設計、モジュラーな構造
- **Local Storage API**: データの永続化
- **File API**: ファイルアップロード処理
- **Drag and Drop API**: ドラッグ&ドロップ機能

## 🔧 開発者向け機能

ブラウザのコンソールで以下の関数を実行できます：

```javascript
// すべての写真を削除
app.clearAllPhotos()

// 写真データをJSONファイルとしてエクスポート
app.exportPhotos()
```
